# Basic-Apache-Server-Security

## Этот репозиторий содержит инструкцию и приложение, которые позволят развернуть безопасный web сервер на apache за пару минут.

## Руководство по активации приложений защиты через Ansible:

### 1. Установка ansible
```
sudo apt update
```
```
sudo apt install ansible -y
```
(Проверить установку можно посмотрев версию)
```
ansible –version
```
___
### 2. Запуск файла protect-server.yml
```
ansible-playbook protect-server.yml --ask-become-pass
```

Запуск сервера:

```
sudo systemctl start apache2
```

Перезапустить сервер можно при помощи команды:
```
sudo systemctl restart apache2
```

Убедиться в функционирования сервера можно посмострев его статус:
```
sudo systemctl status apache2
``` 
- если сервер работает отобразится строка "Active: active (running)"


## По окончанию работы файла (все этапы завершились с исходом ok):
- [X] Установлен и запущен apache2 
- [X] Установлен и запущен fail2ban
- [X] Настроены его правила защиты (block-fast конфиг)
- [X] Установлен и настроен модуль mod_evasive

При ошибке программа выведет её на экран.

Таким образом ваш web сервер защищён от dos и ddos атак (самыен распространённые на сегодня) на уровне web сервера. 
Стабильная работа вашего web приложения обеспечена.
